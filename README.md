**Fraud Detection Using Machine Learning**

This project focuses on building a proactive fraud detection system using financial transaction data.

**Key Highlights:**

Performed exploratory data analysis to identify fraud patterns

Handled extreme class imbalance using **SMOTE**

Built a **Logistic Regression model** achieving ~92% recall

Identified major fraud indicators such as transaction type and balance changes

Proposed business strategies to strengthen fraud prevention

**Tech Stack:**

Python, Pandas, Scikit-learn, SMOTE, Matplotlib, Seaborn
